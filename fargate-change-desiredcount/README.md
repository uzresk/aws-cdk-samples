# Fargate change DesiredCount

 ## Usage
 
 `npm install -g aws-cdk`
 
 create credentials
 
 ```
[default]
aws_access_key_id=
aws_secret_access_key=
region=ap-northeast-1
```

build

`npm run build`

list cdk stack

`cdk ls`

setting config.json

```
cp config.json.sample config.json
```

deploy stack

`cdk deploy`

destroy stack

` cdk destroy`
